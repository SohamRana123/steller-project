🚀 Crypto Subscriptions (Permissionless DApp)

## 📌 Project Description

Crypto Subscriptions is a fully permissionless decentralized application built on the Stellar blockchain using Soroban smart contracts.

It enables anyone to create and subscribe to recurring payment plans without intermediaries. This eliminates the need for centralized subscription services like Netflix or SaaS billing systems.

---

## ⚙️ What it does

* Create subscription plans (price + billing interval)
* Subscribe to any plan freely
* Track next payment timestamps on-chain
* Enable recurring payments via smart contracts
* Remove centralized control from subscription systems

## 🚀 How to Use

1. Deploy the contract on Soroban

2. Create a plan:

```
create_plan(price, interval)
```

3. Subscribe:

```
subscribe(plan_id)
```

4. Trigger payment:

```
pay(subscription_id)
```

---

## 🧠 Core Idea

This DApp is **completely permissionless**:

* Anyone can create plans
* Anyone can subscribe
* Anyone can trigger payments

No central authority controls the system.
